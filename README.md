# QEnegy
前后端分离构建移动端大学生规划软件

基本功能：
- 番茄时钟
- 心情日记
- 账单统计
- 讨论交流

# 开发环境版本校准

```node -v``` v8.11.2

```npm -v``` 6.2.0

```vue``` 2.5.2
