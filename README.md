# QEnegy
weex构建移动端大学生规划软件

基本功能：
- 番茄时钟
- 心情日记
- 账单统计
- 讨论交流

# 开发环境版本校准

```node -v```

v8.11.2

```npm -v```

6.2.0

```vue-cli```

2.9.5

```iview```

2.14.3

```iview-loader```

1.2.1

项目初始化

```vue init webpack QEnergy```

加载依赖

```npm i```
